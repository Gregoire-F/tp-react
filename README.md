# TP React

## Stack technique ##


- React 19 + TypeScript 6
- Vite 8
- Tailwind CSS 4
- Redux Toolkit
- React Router 7
- ESLint


## API ##
L'application se connecte à https://badger.arcplex.dev/api/v2/ (configuré dans .env).


## Fonctionnalités ##

- Authentification — Connexion avec JWT, routes protégées
- Dashboard — Vue d'ensemble des serveurs
- CRUD Serveurs / VMs / Users — Création, modification, suppression avec contrôle d'accès basé sur les rôles
- Clients & Services — Consultation (lecture seule)
- Navigation — Barre de navigation avec déconnexion
